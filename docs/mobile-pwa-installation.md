# Mobil PWA kurulumu

## Android Chrome

HTTPS adresini açın, tarayıcı menüsünden **Uygulamayı yükle/Ana ekrana ekle** seçin. Standalone açılışı, kamera iznini, offline ekranını ve yeni sürüm bildirimini kontrol edin.

## iOS Safari

Safari'de Paylaş → **Ana Ekrana Ekle** adımlarını kullanın. Chrome iOS kurulum yeteneği sunmadığında uygulama içi rehber Safari'ye yönlendirir. Kurulum sonrası safe area, alt navigasyon, klavye ve kamera/galeri seçimini doğrulayın.

Yerel taslaklar cihaz/tarayıcı profilinde ve kullanıcı+organizasyon anahtarıyla saklanır; dosya ve token içermez. Ortak cihazlarda çıkış yapın. Gerçek Android/iOS cihaz testi release kabulünde ayrıca kaydedilmelidir.
